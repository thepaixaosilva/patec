<h2 id="about">📌 Sobre</h2>

Este projeto fez parte do Projeto Integrador do 4° semestre do curso de Análise e Desenvolvimento de Sistemas da Fatec Indaiatuba. A Patec (Plataforma de Avaliação das Turmas EaD com Correção Automática) trata-se de um modelo virtual de folha de respostas, em que os alunos registrarão suas respostas e, após as enviarem, o sistema corrigirá a prova automaticamente, salvando as respostas e a nota do aluno em um banco de dados.
